      GetLocalIPAddressInNetwork Given a remote IP address and a subnet mask, tries to find the local IP address of this host in the same subnet. This is useful to figure out which IP this host is using in the same network as the remote. Inspired by https://stackoverflow.com/a/6803109/1934487
      ValidateIPv4 Returns true if input it is a valid IPv4 address.
      ValidateIPv4Port Returns true if input it is a valid IPv4 address + port, like "127.0.0.1:7000"
