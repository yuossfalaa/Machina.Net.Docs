      ToolCreate(String, Point, Orientation) Method

      Create a new Tool object by defining the Position and Orientation of the Tool Center Point (TCP) relative to the Tool's base coordinate system. In other words, if the Tool gets attached to the robot flange in XYZ [0, 0, 0], where is the tooltip and how is it oriented?
