      Robot Events

      Events

      ActionExecuted Will be raised whenever an Action has completed execution on the device.
      ActionIssued Will be raised whenever an Action has been successfully issued and is scheduled for release to the device or compilation.
      ActionReleased Will be raised whenever an Action has been released to the device and is scheduled for execution.
      MotionUpdate Will be raised whenever new information is available about the real-time information about the state of the device.
